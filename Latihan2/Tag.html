Bagian Tag dan Struktur Tag

Bagian Head

  • Judul Halaman :
    < title > < /title >
  • CSS style :
    < style > < /style >
  • JavaScript :
    < script > < /script >
  • Metadata :
    < meta > < /meta >

Contoh Penggunaan Metadata : 

        <head>
            <meta charset="UTF-8">
            <meta name="description" content="Tag-Tag di HTML">
            <meta name="keywords" content="HTML,CSS,JavaScript">
            <meta name="author" content="Armand Maulana">
        </head>

Bagian Body

  • Teks :
    <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <p>
  • Pendukung Teks :
    <br>, <hr>, <em>, <strong>
  • Gambar :
    <img>
  • Hyperlink :
    <a>
  • List (bullets & numering) :
    <ul>, <ol>, <li>, <dl>, <dt>, <dd>
  • Table :
    <table>, <thead>, <tbody>
  • Form :
    <form>, <input>, <select>, <option>, <button>
  • Script :
    <script>
  • Object :
    <object>
  • Grouping :
    <div>, <span>

Komentar :
<!-- Tag untuk komentar -->
